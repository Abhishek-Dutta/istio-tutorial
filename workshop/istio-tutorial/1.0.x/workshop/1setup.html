Setup

Prerequisite CLI tools

You will need in this tutorial:

  • openshift

    • Mac: brew install openshift-cli

    • Linux/Windows: Download from OKD github page

  • Istioctl

    • Follow the instructions at Istio page

  • Apache Maven

    • Mac OS

    • Fedora: dnf install maven

  • stern

  • siege

    • MAC OS: brew install siege

    • Fedora: dnf install siege

Setup environment

Login in the cluster with the following command:

export WORKSHOP_USER=<your number>
oc login https://master.testans-def0.openshiftworkshop.com -u user$WORKSHOP_USER -p openshift

Example:
export WORKSHOP_USER=1

Istio installation

The environment at https://master.testans-def0.openshiftworkshop.com already contains Istio installed. However, if you want to learn how to install istio on your OpenShift cluster, refer to the document at Upstream Istio installation
